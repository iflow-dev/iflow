7. SVG Legend - Border Numbers & Area Letters

🎯 Reference System for Discussion

Border Numbers (1-4): From outside to inside
1: Dotted outline, 2: Filter state, 3: Button, 4: Icon
🎨 Color-Coded Border Reference:
● Border 1 (Black) = Dotted outline
● Border 2 (Blue) = Filter state border
● Border 3 (Teal) = Button border
● Border 4 (Blue) = Icon border
Area Letters (A-D): Functional areas
A: Border area, B: Content area, C: Button area, D: Icon area
💡 How to Use This Reference System:
  • "Border 1 of area 2" → The dotted outline of the active filter wrapper
  • "Area B of area 1" → The content area of the inactive filter wrapper
  • "Border 3 of area C" → The button border of the button area
  • "Area D of area 2" → The icon area of the active filter wrapper
